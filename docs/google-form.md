# Google Form 設計 + 作成・埋め込み手順

ミチップのウェイティングリストは Google Form で受ける（バックエンド無し・無料・回答はスプレッドシートに蓄積）。

## フォーム設計

- タイトル: `ミチップβ事前登録`
- 説明文（コピペ用）:

  > ピンを立てて、道なりにつなぐ。ロングドライブのための旅程記録サービス「ミチップ」の事前登録フォームです。
  > β公開時に、登録いただいた方から順にご招待します。メールアドレスはβの招待と開発報のお知らせにのみ使います。

### 設問（この順で作成）

| # | 設問 | 形式 | 必須 | 選択肢 |
|---|------|------|------|--------|
| 1 | メールアドレス | 記述式（短文） | **必須** | — ※回答の検証: 「メールアドレス」を設定 |
| 2 | 主にどんな旅をしますか？ | ラジオボタン | 任意 | 車中泊・ロングドライブ / 日帰り・プチ遠征 / 宿泊メインのゆったり旅 / まだこれから |
| 3 | 今いちばんピンを立てたい対象は？ | ラジオボタン | 任意 | 47都道府県 / 市区町村まで細かく / 道の駅 / 温泉 / 城・史跡 / 滝・絶景 / その他（記述） |
| 4 | ミチップに期待すること・欲しい機能があれば | 記述式（段落） | 任意 | — |

設計意図: メールだけなら10秒で完了。2〜3問目はβの機能優先度（制覇の粒度・スポット種別）の判断材料、4問目は濃い初期ファンの声の収集。**これ以上増やさない**（登録率が下がる）。

### フォーム設定

- 設定 → 回答: 「メールアドレスを収集する」は **オフ**（設問1で取るため。オンにすると Google ログインが必須になり離脱が増える）
- 設定 → プレゼンテーション:
  - 確認メッセージ: `登録ありがとうございます。β公開時に、このメールアドレス宛にご招待をお送りします。それまでの開発の様子は X で公開しています。`
  - 「別の回答を送信するためのリンクを表示」= **オフ**
- テーマの色: `#F0A63A`（LP のアクセント色に合わせる）、背景は淡色

## 回答→スプレッドシート連携

回答タブ → スプレッドシートのアイコン →「新しいスプレッドシートを作成」。以後、全登録が自動で1行ずつ入る。件数もここで見る。

## LP への埋め込み

1. フォーム右上「送信」→ `<>`（埋め込む）タブ → 表示された iframe の **src の URL だけ** をコピー
   （形式: `https://docs.google.com/forms/d/e/XXXX/viewform?embedded=true`）
2. `src/lp-src.html` 内の `GOOGLE_FORM_EMBED_URL`（**2箇所**: iframe の src と「新しいタブで開く」リンク）を置換
3. `node src/build.mjs` で `public/index.html` を再生成 → commit & push
4. 実機で: フォームが枠内に収まるか確認。縦に見切れる場合は `src/lp-src.html` の `.form-embed iframe{height:900px}`（モバイルは 960px）を調整して再ビルド

## 運用メモ

- 登録数の定点観測はスプレッドシートの行数で（週1でSNS運用の振り返りと一緒に見る）
- β招待時は BCC 一斉送信でもよいが、件数が増えたら配信サービス（無料枠のあるもの）への移行を検討
