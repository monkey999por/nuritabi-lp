# SNS 運用プラン（X / Instagram・ローンチ20日間・画像込み）

目的: LP への流入 → 事前登録。投稿では「スポットを立てる」「道順でつなぐ」「旅行単位・日単位で残る」を一貫して押し出す。

- ペース: **毎日1本×20日**（X は毎日、IG はフィード6本+ストーリーズ2本を織り込み）。予約投稿でまとめて仕込む
- 画像: すべて **`social/out/` に生成済み**（LPの5デザイン言語で統一）。X 用 1600×900 / IG フィード 1080×1080 / ストーリーズ 1080×1920
- 画像の再生成・修正: `social/cards/dayNN-*.html` を編集 → `node social/build.mjs`（playwright が必要。`PLAYWRIGHT_DIR=<playwright入りのディレクトリ>` で指定可）
- 基本ハッシュタグ: `#ミチップ` を全投稿に。＋投稿ごとに `#個人開発 #車中泊 #道の駅 #ドライブ好きな人と繋がりたい` から1〜2個
- LP: `https://michip-lp.pages.dev/`（`?v=N` でデザイン固定。1=レトロ 2=旅ノート 3=ツッコミ 4=標識 5=HUD）
- **◯◯人 は投稿時に実数へ置換**（Day10/Day20）。数字は盛らない
