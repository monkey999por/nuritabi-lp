# 投稿用テキスト（コピペ完結・タグ込み）

1投稿=1ファイル。中身をそのままコピーして、対応する画像（`../out/` の同名PNG）を添付して投稿する。

- `dayNN-x.txt` … X用（画像: `out/dayNN-x.png`）
- `dayNN-ig.txt` … Instagramフィード用キャプション（画像: `out/dayNN-ig.png`。day06/11/18 はカルーセルで `out/dayNN-ig-1..3.png` の3枚を順に添付）
- `dayNN-st.txt` … ストーリーズの組み立てメモ（画像: `out/dayNN-st.png`）

注意:
- **day10-x / day20-x の「◯◯」は投稿前に実数へ置換**（登録者数はGoogle Formのスプレッドシートの行数）
- 投稿日・時刻・順番は `docs/social-content.md` の20日カレンダーを参照
